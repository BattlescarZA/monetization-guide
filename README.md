### **Step 1: Create GitHub Repository**

1. Go to **github.com**
2. Click **"New Repository"**
3. Repository name: `monetization-guide`
4. Visibility: **Public** (required for GitHub Pages)
5. Description: "Monetization Guide - Make $10,000/Month Online"
6. Click **"Create repository"**

### **Step 2: Upload Files**

**Option A: Using GitHub Desktop (Recommended)**
1. Download GitHub Desktop
2. Click **"Add existing project to GitHub"**
3. Select folder: `C:\Users\ridis\.dominusprime\projects\monetization\blog\gh-pages`
4. Repository name: `monetization-guide`
5. Repository location: Your new GitHub repo
6. Click **"Commit to 'main'"**
7. Click **"Publish repository"**

**Option B: Using Git Command Line**
```bash
cd C:\Users\ridis\.dominusprime\projects\monetization\blog\gh-pages
git init
git add .
git commit -m "Initial commit - Monetization Guide Blog"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/monetization-guide.git
git push -u origin main
```

### **Step 3: Enable GitHub Pages**

1. Go to your repository
2. Click **"Settings"**
3. Click **"Pages"** (left sidebar)
4. Under **"Build and deployment"**:
   - **Source:** Deploy from `main` branch
   - **Folder:** `/ (root)`
5. Click **"Save"**

### **Setup Custom Domain:**

1. Purchase domain (e.g., monetizationguide.com)
2. In GitHub Pages Settings:
   - Click **"Add custom domain"**
   - Enter your domain
   - Follow DNS instructions

### **DNS Configuration:**

Add these records to your domain DNS:

**For A Record:**
```
Type: A
Name: @
Value: 185.199.108.153
```

**For CNAME:**
```
Type: CNAME
Name: www
Value: yourusername.github.io
```
